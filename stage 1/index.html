Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them,whereas an inline element does not start on a new line and only takes up as much width as necessary.

part 2: Structured Document

What is structured document?

A structured document is an electronic document where some method of embedded coding, such as mark-up, is used to give the whole, and parts, of the document various structural meanings according to a schema. A structured document whose mark-up doesn't break the schema and is designed to conform to and which obeys the syntax rules of its mark-up language is "well-formed".The most widely used markup language, in all its evolving forms, is HTML

Boxes to HTML

The HTML div tag is used for defining a section of your document. With the div tag, we can group large sections of HTML elements together and format them with CSS.we use <div></div> to create an invisible box around the document. The difference between the div tag and the span tag is that the div tag is used with block-level elements whilst the span tag is used with inline elements. The Tree-like structure comes from the fact that HTML elements can have other elements inside of them. You can draw this relationship like a family tree. My mother had multiple children. So did her mother, and so on In a browser, this structure shows up as a series of nested boxes. There are boxes inside of boxes inside of boxes, and so on

HTML and DOM

HTML-hypertext markup language DOM-document object model now html:tag->element in tree also dom:tag->element in tree browser turn that 'tag' into elements in tree,it knows how to do it because of DOM,which is a standard convention for representing and interacting elements in html.

what is CSS?

css stands for cascading style sheets,it is used to add style to our html file,CSS file is linked to the main html using" <link ref="stylesheet" href="style.css">.
what does 'cascading' means?
Cascading means Rules(whatever the rules are in css) are applied not only to the elements they directly match but also to all elements,child elements,however if child element has more overcoming rules than the more specific rules takes effect.
